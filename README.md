## ⚗️ Installation

Clone the project

```bash
  git clone https://github.com/rupam-seal/portfolio-2023.git
```

<br>

Go to the project directory

```bash
  cd portfolio-2023

```

<br>

Install Dependencies

```bash
  npm install

```

<br>

Run server

```bash
  npm start
```
